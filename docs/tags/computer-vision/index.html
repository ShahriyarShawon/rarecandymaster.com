  • Automatic TV Show Intro Detection

    July 23, 2025

    Tags: python computer-vision

    Summary: TLDR Using MS-SSIM (Multi Scale Strucutural Similarity ) to calculate the similarity between a known frame and the last frame of an intro in DBZ Kai allowed me to figure out where the intro sequence was in a subset of the pre Boo Saga DBZ Kai episodes. The Single-Core performance operates at about 33 fps and using 4 cores to parallelize the process provides a small increase in performance to 52 fps
